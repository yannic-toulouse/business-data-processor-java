# Business Data Processor

## Description
Business Data Processor is a Java-based console application designed to import, validate and process business order data.  
The project simulates typical enterprise data processing workflows such as file imports, business rule validation and reporting.

This project is developed as a personal learning project with a focus on clean architecture and business-oriented data processing.

## Project Status
**In Development**

Core functionality is currently being implemented.
Active development is ongoing.

## Current Features (Implemented / In Progress)

- CSV file import
- Object mapping (CSV -> Java objects)
- Data validation framework
- Basic business rule processing
- Console logging

## Planned Features

- Configurable business rules
- Automatic report generation
- Configuration file support
- Extended error handling
- Unit test coverage

## Tech Stack

- Java
- Maven
- Git

## How to Run (Development Version)

1. Clone the repository:
   ```sh
   git clone https://github.com/yannic-toulouse/business-data-processor-java.git
   ```
2. Build the project:
   ```sh
   cd data-processor
   mvn clean package
   ```
3. Run the application:
   ```sh
   java -jar target/data-processor-0.1.jar
   ```
